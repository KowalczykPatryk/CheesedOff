# CheesedOff
Projekt zrealizowany jako aplikacja typu SPA (Single Page Application) z wykorzystaniem biblioteki React oraz języka TypeScript. Część serwerowa w Django z użyciem Django REST Framework i łączy  się z relacyjną bazą danych PostgreSQL. Uwierzytelnienie  zrealizowane z użyciem tokenów JWT. Interfejs  zgodny ze standardem HTML5 i walidowany przez W3C.
